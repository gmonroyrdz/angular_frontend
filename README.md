# Empleados Frontend (Angular + Angular Material)

## Requisitos
- Node.js 18+ (recomendado)
- npm
- Angular CLI (opcional, instalar globalmente `npm i -g @angular/cli`)

## Instalación
```bash
npm install
```

## Ejecutar en modo desarrollo
```bash
npm start
```
Esto lanzará `ng serve` y abrirá http://localhost:4200.

## Notas
- El backend esperado está en http://localhost:8080 (configurable en `src/environments/environment.ts`)
- Angular Material ya está referenciado; tras `npm install` puede ser necesario ejecutar `ng add @angular/material` para completar temas y fonts, o copiar estilos manualmente.
